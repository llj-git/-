一、网站功能
随着电子设备的普及，眼睛的压力也急剧增大。青少年甚至儿童的近视率不断攀升，因此爱眼护眼刻不容缓。
本网站通过护眼教程，护眼小知识。同时可以在网站上做眼保健操等，还可以使用测视力功能检测自己的实力。
二、网站整体规划
网站整体采用简约风格，主要分为网站主页、眼保健操、测视力、护眼小知识四个板块。
三、网站技术及页面功能
网站整体用到Bootstap栅格系统的响应式布局附加媒体查询，元素宽度尽量采用百分比设置。

首页上顶部为响应式nav，选项元素用伪元素after来添加边框样式。主题内容为两个轮播图。
右侧有个报时区，会根据时间的不同变换不同的问候语。最右侧两个按钮采用了hover添加了旋转样式。

眼保健操页面上方是个视频播放器。下拉为眼保健操教程，都是用伪元素撑开div属性再添加背景图片，大小可随屏幕变化。右侧为回到底部小按钮。

测视力页面 采用了js来计算视力值，左右按钮可以调试视力图的大小。用随机数随机生成视力图的方向，按钮都会触发。

护眼小知识页面 三个小知识例子，其中每个小知识对应一个小知识页面
